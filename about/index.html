网站前言

这里是九依，一个热爱前端的的设计师，这个网站寄托了我对世界中所有的向往，世界无垠大，我想去看看…

网站说明

  • 本网站采用多线路并行部署，一份源码同时分发给多个托管商，确保能够 24 小时进行访问

  • 主域名：ellisy.top | jiuyi.ellisy.top

    解析至 Vercel，Vercel 有全球的泛播节点，兼具全球 CDN 功能，国内访问速度较快

  • 备用域名 1: jiuyii.ellisy.top

    解析至 Github, Github 是全球最大开源网站, 作为网站自部署中转使用 ,国内访问速度不确定

  • 备用域名 2: rai.elliay.top

    解析至 Render，一个免费容器托管平台，节点支持全球泛播，但是国内速度一般

  • 备用域名 3: cloud.ellisy.top rai.elliay.top

    解析至 cloudflare，Cloudflare 也具有全球泛播节点，其为一家位于美国的跨国科技企业，总部位于旧金山，国内速度一般

    ellisy.top 主域名使用腾讯云进行 CDN 解析，暂时未作分流处理，后面会提上日程（应该？）

  • 线路均支持 HTTPS 协议

技术栈&框架

  • 博客框架：Hexo 6.2.0

  • 主题：Butterfly 4.3.1

  • 源码仓库：Github

  • 托管与部署：本网站采用多线部署，包括Vercel + Cloudflare + Railway + Github Pages 等

  • 评论系统：Twikoo系统 + Vercel部署 + MongoDB提供存储服务

  • 图床：缤纷云 Bucket + 初柒工坊图床

  • 静态资源压缩: gulp + JS & CSS Minifier (Minify)

  • 自动部署：Github Action

  • SEO 优化：谷歌 + 必应 + 百度

  • 音乐平台：hexo-tag-aplayer + MetingJS

  • 异步加载与缓存加速技术：Pjax

  • 弹窗框架支持：Element + Vue

  • 开发语言：[HTML] + [CSS3] + [JavaScript] + [Pug] + [YML]等

  • 线路优化：利用Service Worker劫持主域链接至缤纷云 Bucket，从而实现将请求分流至国内线路，以提高访问速度(参考 LYX 大佬的方案)

关于网站

站长这个名字有什么含义？：“长安九陌，通向四方。“9”是科学中的数字，但是对于数字却也是终点，文明的发展是浪漫的，在繁星中寻求真相，在无垠宇宙中探索未知。文明需要自己文化的传承，于是便有了九。绿隐于山，白云深处。直须看，归来不惮远。灵感的源泉常常在于对过往的深思与对未来的渴望。在这片无垠的天地中，我愿意做一个追梦者，依依不舍地向前迈进。
用途：网站是独属于自己的一个名片，在上面写写东西，记录生活，分享知识，把自己想与他人展现的技能放在上面，何尝不是一件有趣的事情！
想法：得益于 Hexo 对于 html 的编写更加自由，同时部署简单且可以最低成本的去运营，这里也感谢一些 Hexo 主题魔改的开源大佬的教程，使咱可以在短时间就部署出这么好看的网站，我会在这里分享一些技术开发教程，当然啦，还有艺术画作。

格言

在花海中找寻故事，每一朵花皆是生命的篇章，承载着无数个体的悲欢离合；用漫天繁星绘画世界，星海闪烁如梦，映照出人心深处的渴望与挣扎。我们在这片丰富的花海中，与每一个灵魂相遇，感悟彼此的苦乐与希望。当历史的尘埃渐渐沉淀，唯有那些真实的情感与经验，才会如星光般闪耀。愿我们在这微小的舞台上，珍视每一个故事，追寻那弥足珍贵的连结，书写出属于人性的辉煌乐章。
